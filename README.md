# Fatiha Imahoui — Personal Portfolio

A personal portfolio website built from scratch with pure HTML, CSS, and JavaScript. Designed to showcase my projects, skills, experience, and background as a first-year Software Engineering student at FST Errachidia.

🔗 **Live site:**  [fatiha-portfolio.netlify.app](https://fatiha-portfolio.netlify.app/)

---

## Built With

- **HTML5** — semantic structure
- **CSS3** — Flexbox, Grid, custom properties, animations, responsive design
- **Vanilla JavaScript** — typing effect, scroll reveal, mobile menu, active nav, form handling
- **Boxicons** — icon library
- **Google Fonts** — Poppins typeface
- **Formspree** — contact form email delivery

---

## Run Locally

No build tools or dependencies needed. Just clone and open:

```bash
git clone https://github.com/fatiha-alt/Portfolio.git
cd portfolio
```

Then open `index.html` in your browser. That's it.
